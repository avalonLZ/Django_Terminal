由于想继续学习服务器、数据库等相关知识，所以自己又在VPS上用Python做了协议解析，并入库。最后使用Django框架将数据展示在Web上
Django版本:1.11.2

1、MQTT_Data.sql为MySQL数据库的备份文件，想要尝试此项目需要先安装mysql
